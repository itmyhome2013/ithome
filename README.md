### 项目框架
===========

#### 使用技术

+ EasyUI - 1.3.6
+ Bootstrap - 3.3.1
+ jQuery - 1.8.0
+ Struts2 - 2.3.14
+ Spring - 3.1.3
+ Hibernate - 4.1.12
+ JDBC
+ Oracle - 10g
+ Activiti - 5.1.3
+ Spring Date MongoDB - 1.4.1
